# Storybook + React + Flow + Babel & Webpack

## Purpose

Understand how to configure Storybook.js with a React App built with Babel & Webpack + using Flow types.

## Instructions

### Installation

Run `yarn install`.

### Webpack Dev Server

Run `yarn start`.
