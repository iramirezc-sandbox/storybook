# Storybook + React + Flow + Babel & Webpack example

## Purpose

Understand how to configure Storybook.js with a React App built with Babel & Webpack + using Flow types.

## Instructions

### Installation

Run `yarn install`.

### Webpack Dev Server

Run `yarn start`.

### Stories

Run `yarn storybook`.

Run `yarn storybook --debug-webpack` to output webpack config.
